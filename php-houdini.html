PHP Houdini

PHP Houdini is a PhpStorm and plugin that adds configurable autocompletion for classes using magic methods and properties.

Why?

  • Highly Configurable Use a config file to describe your classes that have magic methods. No menus to slog through.
  • Self-documenting API Use PHP to describe your magic classes. Configure your classes with a subset of PHP that has autocompletion for itself - so you can use PhpStorm to aid you in configuring your own specialized autocompletion. No JSON API or specialized DSL to learn that you'll just forget later.
  • With Types Navigate to classes of the magic methods or magic properties because PHP Houdini includes type information for your magic properties or magic methods.
