PHP Houdini

PHP Houdini is a PhpStorm and plugin that adds configurable autocompletion for classes using magic methods and properties.

Demo

How?

  • Use a config file to define your classes with magic methods
  • Use PHP to describing your magic classes. It's configured with a subset of PHP (like .phpstorm.meta.php) that has autocmpletion for itself - so you can use PhpStorm to aid you in configuring your autocompletion. No JSON API or any specialized DSL to learn.
  • Navigate to classes of the magic methods or properties because PHP Houdini includes type information.
